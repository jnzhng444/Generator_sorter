Para utilizar el generador:
Se escribe en la terminal .\generator --size <SIZE> --output <OUTPUT FILE PATH>
Donde size: puede ser SMALL, MEDIUM, LARGE
    output: nombre del archivo con la extension deseada

Para utilizar el sorter:
Se escribe en la terminal .\sorter -input <INPUT FILE PATH> -output <OUTPUT FILE PATH> -alg <ALGORITMO>
Donde input: nombre del archivo binario a ordenar
      output: nombre del archivo ordenado con la extension deseada
      alg: puede ser QS, BS, IS
